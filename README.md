# life-gleaner
此仓库是收藏夹，储存所有我公开的文字。我不敢妄自称之为“文学”，毕竟很多文字都不完美，于是以“拾穗”作为仓库的标题，毕竟现实是创作的土壤，文字是生活的“拾穗”。衷心希望诸君能于此享受阅读的乐趣。

### 创作理念
- Écrire pour se comprendre et secourire.
- Vivir para contarla.
- 暂时谨以Gabriel García Márquez在*Vivir para contarla*（译作“活着为了讲述”）的陈述解释：生活不是我们活过的日子，而是我们记住的日子，我们为了讲述而在记忆中重现的日子。
