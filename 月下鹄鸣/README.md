# 月下鹄鸣
中学时代的总结与终结。

### 自序
> OH~ 我有我心底故事，亲手写上每段得失乐与悲与梦儿…… ---Beyond,*不再犹豫*,梁美薇词

> ……将这样的感触 写一封情书 送给我自己 感动得要哭 很久没哭 不失为天大的幸福…… ---王菲,*笑忘书*,林夕词

“……有些情感，有些思想，只能深埋心底，无法吐露，却也不会消逝；它们如雪花般堆积，终成一片白茫茫的孤独大陆——在那里，任何足迹终将消隐。……”我在高考前127天写下这段文字，然而当时的我怎么也不会料到，雪花终将融化，化作洪水冲毁心灵的堤坝，在150天后公之于众。

我从记忆里“抄”出了这本书。于此窥见一个浪漫主义而又现实主义的、高尚而又低俗的、真诚而又虚伪、坚强而又软弱的矛盾人格，藉此为平平无奇而又“波澜壮阔”的高中生活、为青春期的悸动与美好留下一份“历史底稿”。

这是一部杂文集，有散文、小说，还有论述。书中的真真假假也无需关注，正道是：

*假作真时真亦假，无为有处有还无。*

写作使我快乐，我将这段文字分享，惟怨自己才疏学浅，文字上必有诸多不善之处，但还希望诸君能够喜欢。
